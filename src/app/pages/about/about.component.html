Route Module = App Module

Dentro del core module por ejemplo vamos a ver toda la logica de los servicios http, porque especialmente estos se manejan y distribuyen estre todos

Shared Module

Practicamente es un modulo que contiene todo lo que vayas a reutilizar y a diferencia del Core que nada se exporta aquí si se exportá el modulo

Helpers

Es una función independientemente de angular que puede ejecutar ciertas acciones, por ejemplo puede ser por ejemplo una función que capitalice palabras, un sum o un calculate-percentage

Componentes compartidos y de estructura

Estos no son iguales porque para los componentes de presentación o de estructura generalmente para ello se crea un modulo llamado layout

Notas

  • Los dumbs no deben hacer peticiones http.
  • Lo deben tener acceso a los servicios.
  • No deben almacenar nada en backend
  • No deben hacer absolutamente nada mas que encargarse de presentar el renderizado, ya que dicho de otra forma es el lienzo donde se va a pintar.

Si se quiere hacer logica se empezaría a abstraer en subcomponentes llamados smarts, en este proyecto hay 2 smarts que son el list y el card
